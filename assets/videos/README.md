# Videos

把 Wan2.2 / Seedance 等视频样本放在对应子目录。

目录对应：

```text
assets/videos/puppet/  木偶动画
assets/videos/long/    长视频
```

推荐命名：

```text
puppet/wan22-demo.mp4
puppet/wan22-demo-cover.webp
long/bilibili-case-cover.webp
```

然后在根目录 `portfolio-data.js` 的 `videos` 数组里添加对应条目。

如果视频已经发布到 B 站，也可以只放封面图，然后在 `portfolio-data.js` 里填 `url`。
